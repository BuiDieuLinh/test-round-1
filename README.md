# Test Round 1

## Giới thiệu các bài test

- **Test 1 - Phân tích văn bản**: Phân tích file văn bản để đếm số từ duy nhất và tìm top 3 từ xuất hiện nhiều nhất bằng Web Worker.
- **Test 2 - Trực quan hóa hứng nước mưa**: Tính toán lượng nước mưa tích lũy và vẽ mô hình 2D trên Canvas.
- **Test 3 - Hệ thống đặt bàn nhà hàng**: API đặt bàn sử dụng Transaction để chống trùng lịch đặt và hỗ trợ tự động hủy đơn sau 15 phút.
- **Test 4 - Hệ thống lưu trữ viễn thông**: Thiết kế sơ đồ quan hệ (ERD) và viết các câu lệnh truy vấn SQL thống kê dữ liệu cuộc gọi.
